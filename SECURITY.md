# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via email to vilker.pessin@gmail.com.

**Please do not report security vulnerabilities through public GitHub issues.**

When reporting, please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

## Response

You can expect a response within a few days. We will:

1. Confirm receipt of your report
2. Investigate and assess the issue
3. Work on a fix if confirmed
4. Credit you in the fix (unless you prefer to remain anonymous)
